# 🚀 Real-Time System Resource Monitoring Dashboard

> "A professional DevOps monitoring solution developed using Python and Flask to visualize real-time hardware performance metrics."

## 📌 Project Overview & Objectives
* **Infrastructure Health:** To provide a visual interface for monitoring server health.
* **Resource Optimization:** Identifying CPU and Memory spikes.
* **Asynchronous Learning:** Demonstrating AJAX to update UI.

## 🛠️ Technical Stack
* **Language:** Python 3.x
* **Framework:** Flask 
* **Monitoring Library:** `psutil` 
* **Visualization:** Plotly.js 
* **Communication:** AJAX / Fetch API 

## 🚀 Deployment Results
The dashboard successfully renders live metrics. The CPU usage displays fluctuations based on active processes, while Memory usage reflects the current RAM load.
